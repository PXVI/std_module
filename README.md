# std_module
------------------------
A simplified one stop RTL library for all the basic Verilog modules.


### Contents ( List of modules )
- [X] Basic Gates
  - [X] AND
  - [X] OR
  - [X] NOT
  - [X] NOR
  - [X] NAND
  - [X] XOR
  - [X] XNOR
- [ ] Logic Modules
  - [X] Multiplexer 2x1
  - [ ] Multiplexer Nx1
  - [X] DeMultiplexer 1x2
  - [X] DeMultiplexer 1xN ( Module Script )
  - [ ] Encoder NxM
  - [ ] Decoder MxN
- [ ] Arithematic Modules
  - [ ] Half Adder
  - [ ] Full Adder
  - [ ] Half Subtractor
  - [ ] Full Subtractor
- [ ] Sequential Modules
  - [ ] Left Shifter ( SISO )
  - [ ] Right Shifter ( SISO )
  - [ ] Cyclic Shifter ( SISO )
  - [ ] Parallel Load Serial Shifter ( PISO )
  - [ ] Barrel Shifter
  - [ ] Linear Feedback Shift Register ( LFSR )
- [ ] Buffer Modules
  - [ ] Synchronous FIFO
  - [ ] Asynchronous FIFO


#### Note :
All the modules here are designed in behavioural format of the RTL code. It is very unlikely that you will find any hardcore structural code in here. If there are any, I'll make sure to mention it in this list here, so its easier to keep track of.
